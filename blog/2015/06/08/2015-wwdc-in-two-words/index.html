2015 WWDC в двух словах

8-06-2015 | Comments

Что нового на WWDC 2015:

  • Будут стримы лаб (для разработчиков, круть)
  • OS X
    • OS X El Capitan
    • Новые жесты
    • Новый spotlighg
    • Разделение рабочей области на две части
    • Производительность
    • Metal для Mac
    • Осенью доступна будет новая система
  • iOS
    • iOS 9
    • Сири новый интерфейс
    • Более умная Сири
    • API для поиска
    • Apple Pay
      • Добавили пару магазинов…
      • Теперь не только Америка но еще и Англия

    • Notes

      Так любил я простоту ноутс, теперь оно навороченное :(

      • можно рисовать пальцем
      • поиск лучше (прямо эвернот какой-то :) )
    • Maps
      • общественный транспорт
      • Сири может искать маршруты
      • может искать локации по типу (даже те места, где можно платить apple pay)
    • News
      • выбираешь источники, темы интересные, получаешь новости
      • удобная навигация
      • обучается, показывает интересные темы
      • есть вкладка для исследования нового

      Интересно, можно ли добавить свой источник?

    • клавиатура как трэкпад, интересненько…
    • Многозадачность
      • реакцию на двойной тап на хоум поменяли
      • можно держать два открытых приложения одновременно
      • можно смотреть видео и одновременно что-то делать (картинка в картинке)
    • Дольше живем на батарейке
    • Специальный режим сохранения батарейки
    • Показали новые фреймворки для игр
    • Swift
      • Swift 2
      • Свифт теперь опенсорс
  • watchOS
    • watchOS 2
    • Можно настраивать главный экран часов (поставить фотоальбом или фото)
    • Добавить любых виджетов
    • Специальный режим во время зарядки
      • спец будильник
    • Много новых фишек в коммуникации, рисование на экране, эппл пэй, Сири и т.д.
    • Обещают что можно будет делать приложения которые будут работать на часах! не только дополнительным экраном для iPhone
    • API для доступа к микрофону, динамику, видео, показателям здоровья, HomeKit, доступ к крутилке на часах
    • Управление машиной (закрыть/открыть машину, поменять настройки климат контроля)
  • One more thing…
    • Apple Music
      • Революционный сервис
      • 24/7 радио
      • Связь фанатов с артистами
      • первые три месяца бесплатно
      • потом 10 дол. в месяц
